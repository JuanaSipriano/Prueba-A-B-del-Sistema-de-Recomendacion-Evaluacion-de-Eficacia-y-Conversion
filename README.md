# Prueba A/B del Sistema de Recomendación: Evaluación de Eficacia y Conversión

Este proyecto analiza si la prueba A/B **recommender_system_test** se realizó de manera correcta y evalúa si el nuevo sistema de recomendaciones logró mejorar el rendimiento del embudo de ventas en comparación con el grupo de control. El análisis abarca la distribución de eventos, la evolución temporal de la actividad y las tasas de conversión en cada etapa del embudo.

## Objetivo del proyecto
- Verificar la validez estadística de la prueba A/B.
- Comparar el comportamiento de los usuarios entre los grupos A (control) y B (experimental).
- Evaluar si el nuevo sistema de recomendaciones aumentó las conversiones del embudo al menos un **10 %**.
- Identificar posibles anomalías o efectos inesperados en el comportamiento de los usuarios.
